# PAN-NUMBER-VALIDATION
A data analytics project using Python and Pandas to validate Indian PAN numbers, clean datasets, detect duplicates, handle missing values, and generate accurate validation results.

# PAN Number Validation

A Python and Pandas project that validates Indian PAN numbers and performs data cleaning on the dataset.

## Features

- Validate Indian PAN numbers using regular expressions (Regex)
- Detect valid and invalid PAN numbers
- Handle missing values
- Remove duplicate records
- Perform basic data cleaning using Pandas
- Export the cleaned dataset

## Technologies Used

- Python
- Pandas
- Regular Expressions (Regex)
- OpenPyXL
